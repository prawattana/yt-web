# YT Downloader

เว็บหน้าเดียวสำหรับโหลดเสียง/วิดีโอจาก YouTube ผ่าน **yt-dlp** โดยไม่ต้องพิมพ์ใน terminal
วาง URL แล้วระบบดึงข้อมูลให้อัตโนมัติ เลือกได้ว่าจะโหลดเป็นเสียงหรือวิดีโอ + เลือกนามสกุลไฟล์เอง

ธีม ThinkPad ดำ-แดง-ขาว

## ต้องมี
- [Node.js](https://nodejs.org) (v18+)
- [yt-dlp](https://github.com/yt-dlp/yt-dlp) บน PATH — `winget install yt-dlp`
- [ffmpeg](https://ffmpeg.org) บน PATH — `winget install ffmpeg` (จำเป็นสำหรับแปลง mp3/รวมไฟล์)

## วิธีใช้
ดับเบิลคลิก **เปิดเว็บโหลดยูทูป.bat** หรือ:
```bash
node server.js
```
แล้วเปิด http://localhost:6200

1. วางลิงก์ YouTube → ข้อมูลคลิปขึ้นเอง (รูป/ชื่อ/ความยาว)
2. เลือกแท็บ 🎵 เสียง หรือ 🎬 วิดีโอ
3. เลือกนามสกุลไฟล์ (เสียง: mp3/m4a/opus/wav/flac · วิดีโอ: mp4/mkv/webm + ความละเอียด)
4. กดดาวน์โหลด → เห็นแถบความคืบหน้า → ไฟล์เด้งลงผ่านเบราว์เซอร์

## หมายเหตุ
- ทำงานบนเครื่องตัวเองเท่านั้น (localhost) ไฟล์ชั่วคราวเก็บใน temp แล้วสตรีมให้เบราว์เซอร์
- พอร์ต 6200 แก้ได้ใน `server.js`
