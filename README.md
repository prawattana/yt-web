# YT Downloader

เว็บหน้าเดียวสำหรับโหลดเสียง/วิดีโอจาก YouTube ผ่าน **yt-dlp** โดยไม่ต้องพิมพ์ใน terminal
วาง URL แล้วระบบดึงข้อมูลให้อัตโนมัติ เลือกได้ว่าจะโหลดเป็นเสียงหรือวิดีโอ + เลือกนามสกุลไฟล์เอง

ธีม ThinkPad ดำ-แดง-ขาว

## ต้องมี
- [Node.js](https://nodejs.org) (v18+)
- [yt-dlp](https://github.com/yt-dlp/yt-dlp) บน PATH — `winget install yt-dlp`
- [ffmpeg](https://ffmpeg.org) บน PATH — `winget install ffmpeg` (จำเป็นสำหรับแปลง mp3/รวมไฟล์)

## วิธีใช้
ดับเบิลคลิก **เปิดเว็บโหลดยูทูป.bat** หรือ:
```bash
node server.js
```
แล้วเปิด http://localhost:6200

1. วางลิงก์ YouTube → ข้อมูลคลิปขึ้นเอง (รูป/ชื่อ/ความยาว)
2. เลือกแท็บ 🎵 เสียง หรือ 🎬 วิดีโอ
3. เลือกนามสกุลไฟล์ (เสียง: mp3/m4a/opus/wav/flac · วิดีโอ: mp4/mkv/webm + ความละเอียด)
4. กดดาวน์โหลด → เห็นแถบความคืบหน้า → ไฟล์เด้งลงผ่านเบราว์เซอร์

## ลิงค์หน้าเว็บ (ส่งให้เพื่อน)
หน้า UI โฮสต์อยู่บน GitHub Pages — เป็นลิงค์เดียวส่งให้ใครก็ได้:

### 👉 https://prawattana.github.io/yt-web/

**สำคัญ:** หน้านี้เป็นแค่ "หน้าจอ" ตัวโหลดจริง (yt-dlp) ทำงานบนเครื่องของผู้ใช้เอง
ดังนั้นเพื่อนที่เปิดลิงค์ต้อง:
1. ติดตั้ง Node + yt-dlp + ffmpeg
2. `git clone` repo นี้ แล้วดับเบิลคลิก `เปิดเว็บโหลดยูทูป.bat` (เปิดค้างไว้)
3. กลับมาที่ลิงค์ GitHub Pages → หน้าเว็บจะเชื่อมกับโปรแกรมในเครื่องผ่าน `http://localhost:6200`

ถ้ายังไม่ได้เปิดโปรแกรม หรือยังขาด yt-dlp/ffmpeg หน้าเว็บจะ **ขึ้นแบนเนอร์บอกวิธี** ให้เอง
(การเชื่อม HTTPS → localhost ใช้ CORS + Private Network Access header ในเซิร์ฟเวอร์)

## ส่งแบบอื่น
- **เพื่อนต่อ Wi-Fi เดียวกัน:** เปิดเซิร์ฟเวอร์ที่เครื่องคุณ ให้เพื่อนเข้า `http://[IP เครื่องคุณ]:6200` (เพื่อนไม่ต้องลงอะไร)
- **ใช้เองล้วนๆ:** เปิด `.bat` แล้วเข้า `http://localhost:6200`

## หมายเหตุ
- ทำงานบนเครื่องตัวเองเท่านั้น (localhost) ไฟล์ชั่วคราวเก็บใน temp แล้วสตรีมให้เบราว์เซอร์
- พอร์ต 6200 แก้ได้ใน `server.js`
